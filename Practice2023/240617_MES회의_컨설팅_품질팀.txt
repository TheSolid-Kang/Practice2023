* 품질그룹 전반적인 업무
1. 팀
가. 선행품질: 개발품질, 수입검사, 출하검사 업무 진행
-> 수입검사: 협력사 납품 검사
-> 개발품질: 개발 시 ISIR 초품인증
-> 출하검사: 현장 작업한 것에 대해 출하할 때 검사 함.
	-> 대부분 기능 위주의 검사 함.
나. 품질보증: 고객사 대응
다. 품질관리: 협력사관리, 현장xx, 기타 개선업무
-> 스킨케어 담당:
	-> 협력사 양산 시 협력사의 역량 향산
	-> 협력사가 양품 만들 수 있도록 가이드 제시
	-> 불량 나왔을 때, 이 제품 사용할 수 있는지 없는지 검증 진행
-> 메이크업 담당:
	-> 협력사 양산 시 협력사의 역량 향산
	-> 협력사가 양품 만들 수 있도록 가이드 제시
	-> 불량 나왔을 때, 이 제품 사용할 수 있는지 없는지 검증 진행

2. 트러블
가. 트러블 발생 시 담당들이 가서 이 제품을 써도 되는지 안 되는지 구분함.
나. 

3. 수입검사 과정
-> 'YQMS.검사성적서' 화면에서 협력사가 성적서판정 정보를 입력한다.
-> 수입검사실에서 검사표준서 출력하여 검사 진행
	-> 해당 표의 CTQ만 검사 함.

4. 품질관리팀의 주요업무
가. 4M 관리
나. 신제품관리

5. MES는 재고 확인할 때 씀.

* 기타사항
1. QC가 현장 돌아다니며 공정검사 진행 함.
2. 품질관리팀은 품질을 기준으로 협력사를 평가하고 점수를 준다.
3. 협력사 평가 점수가 낮다면 패널티가 있다.
-> 단, 거래중단 같은 건 없다.
4. 협력사 평가 점수가 높다면 인센티브가 있다.
5. CJ문의: 불합격 품 발생 시 
	-> 'YQMS.시정예방조치'화면에 협력사가 어떻게 예방할지 대책을 등록해야 한다.
6. 표준견복 제작 방법
가. 고객사에서 연우에 표준견본 전달 받음.
나. 제품경본 만들어서 고객사 컨펌 받은 후 첫생산 한 제품을 표준견본으로 지정.
